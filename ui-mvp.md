## 5. shadcn 컴포넌트 사용 목록

| 컴포넌트 | 사용처 |
|----------|--------|
| Card, CardHeader, CardContent | 각 요약 카드 |
| Badge | 카테고리 뱃지, 타임스탬프 버튼 |
| Button | 요약 시작, 복사, 저장 |
| Input | URL 입력 |
| Progress | 로딩 진행 바 |
| Separator | 섹션 구분선 |
| Table | 영어 단어 표, 육하원칙 |
| Checkbox | 자기계발 체크리스트 |
| Skeleton | 로딩 스켈레톤 |
| Tooltip | 타임스탬프 hover 설명 |

## 6. 타임스탬프 버튼 디자인

```tsx
// 타임스탬프 클릭 → 플레이어 해당 시점 이동
<Badge
  variant="outline"
  className="cursor-pointer hover:bg-primary hover:text-white
             transition-colors font-mono text-xs"
  onClick={() => seekTo(timestamp)}
>
  ▶ {timestamp}
</Badge>
```

클릭 전: 아웃라인 뱃지 (회색)
클릭/호버: 강조 색상 + ▶ 아이콘
