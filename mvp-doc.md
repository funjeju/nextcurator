# 넥스트 큐레이터 — MVP 개발 명세 (Phase 1)

## 슬로건
> "흘러가던 유튜브, 이제 AI가 내 창고에 쌓아준다"

---

## 1. Phase 1 목표 (단 하나)

YouTube URL을 입력하면 AI가 콘텐츠 성격을 자동 감지하고,
용도에 맞는 포맷으로 요약해서 보여준다.

- 상단: YouTube 플레이어 (임베드)
- 하단: 카테고리별 구조화 요약 + 타임스탬프 (클릭 시 해당 시점 이동)

---

## 2. 6가지 요약 카테고리

### 🍳 요리 (Recipe)
| 항목 | 내용 |
|------|------|
| 요리명 | 자동 추출 |
| 재료 | 재료명 + 분량 리스트 |
| 조리 단계 | 단계별 설명 + 타임스탬프 |
| 핵심 팁 | 강조 포인트 2~3개 |
| 난이도/소요시간 | 자동 추출 |

### 🔤 영어 학습 (English Learning)
| 항목 | 내용 |
|------|------|
| 핵심 표현 | 표현 + 한국어 의미 + 예문 |
| 주요 단어 | 단어 + 발음 포인트 + 예문 |
| 반복 패턴 | 영상에서 반복되는 문장 구조 |
| 문화 맥락 | 배경 설명 (해당 시) |
| 타임스탬프 | 표현별 등장 시점 |

### 📐 일반 학습 (General Learning)
| 항목 | 내용 |
|------|------|
| 핵심 개념 | 개념명 + 설명 |
| 구조화 요약 | 목차형 정리 |
| 핵심 포인트 | 3~5개 bullet |
| 예시/사례 | 영상 내 예시 추출 |
| 타임스탬프 | 개념별 등장 시점 |

### 🗞️ 뉴스/정보 (News & Info)
| 항목 | 내용 |
|------|------|
| 핵심 요약 | 3줄 요약 |
| 육하원칙 | 누가/언제/어디서/무엇을/어떻게/왜 |
| 배경 | 이슈 맥락 설명 |
| 시사점 | 핵심 의미 1~2개 |
| 타임스탬프 | 주요 발언/장면 시점 |

### 💪 자기계발/동기부여 (Self-Development)
| 항목 | 내용 |
|------|------|
| 핵심 메시지 | 1줄 핵심 |
| 주요 인사이트 | 3~5개 |
| 실천 체크리스트 | 바로 적용 가능한 액션 |
| 명언/인용 | 타임스탬프 포함 |
| 타임스탬프 | 인사이트별 시점 |

### 🧳 여행/브이로그 (Travel & Vlog)
| 항목 | 내용 |
|------|------|
| 방문지 리스트 | 장소명 + 간단 설명 |
| 동선 | 추천 순서 |
| 실용 정보 | 가격/운영시간/팁 |
| 하이라이트 | 핵심 장면 타임스탬프 |
| 주의사항 | 놓치기 쉬운 팁 |

## 3. 공통 UI 구조

```
[상단] YouTube 플레이어 (16:9 임베드, 전체 너비)
─────────────────────────────────────────
[중단] 카테고리 뱃지 + 영상 제목 + 채널명
─────────────────────────────────────────
[하단] 구조화 요약 카드
  - 카테고리별 전용 포맷
  - 타임스탬프 클릭 → 플레이어 해당 시점 이동
  - 각 섹션 복사 버튼
─────────────────────────────────────────
[최하단] 저장 버튼 (Phase 2 라이브러리 연결 예정)
```

---

## 4. 사용자 플로우

```
1. 홈에서 YouTube URL 입력
2. AI가 자막 추출 + 카테고리 자동 감지
3. 진행 상태 표시 ("영상 분석 중... 요약 생성 중...")
4. 요약 결과 화면 표시
5. 타임스탬프 클릭으로 영상 탐색
6. [저장하기] → Phase 2에서 라이브러리 연결
```

---

## 5. Phase 2 이후 확장 로드맵 (개발 시 확장성 고려)

| Phase | 핵심 기능 |
|-------|----------|
| Phase 1 | URL 입력 → AI 요약 → 타임스탬프 뷰 |
| Phase 2 | 회원가입 + 라이브러리 저장 + AI 대화 검색 |
| Phase 3 | AI 영상 탐색 + 블로그 발행 레이어 |
| Phase 4 | 광장(THE SQUARE) + 집단 지성 레이어 |

---

## 6. 성공 기준 (Phase 1)

- URL 입력 후 요약 결과까지 30초 이내
- 6개 카테고리 자동 분류 정확도 90% 이상
- 타임스탬프 클릭 → 플레이어 이동 정상 동작
- 모바일/데스크톱 반응형
