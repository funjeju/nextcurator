## 1. 서비스 개요

**쏙튜브**는 유튜브 영상·음성 녹음·PDF 문서를 AI로 분석해 콘텐츠 유형에 맞는 구조화된 요약을 자동 생성하고, 학습·지식 관리·협업 교육까지 연결하는 AI 지식 큐레이션 플랫폼입니다.

| 항목 | 내용 |
|---|---|
| 기술 스택 | Next.js (App Router), Firebase, Gemini 2.5 Flash, Claude (Anthropic), TypeScript, Tailwind CSS |
| 기본 테마 | 다크 모드 (라이트 전환 가능) |
| 지원 환경 | 모바일 / 태블릿 / 데스크톱 (반응형) |
| AI 엔진 | Google Gemini 2.5 Flash (전사·분류·요약), Claude (Anthropic) (범주별 심층 요약), text-embedding-004 (벡터 검색) |

## 2. 콘텐츠 입력 및 AI 분석

### 2-1. 유튜브 URL 분석

- 홈 화면 URL 입력창에 유튜브 주소 붙여넣기
- 영상 ID 추출 → 메타데이터(제목·채널·길이·게시일) 자동 획득
- 자막(자동 생성 포함) 전문 추출
- Gemini로 **9개 카테고리 자동 분류**:

  | 카테고리 | 아이콘 | 설명 |
  |---|---|---|
  | recipe | 🍳 | 요리·레시피 영상 |
  | english | 🔤 | 영어 학습 |
  | learning | 📐 | 학습·강의 |
  | news | 🗞️ | 뉴스·시사 |
  | selfdev | 💪 | 자기계발 |
  | travel | 🧳 | 여행 |
  | story | 🍿 | 스토리·영화·드라마 |
  | tips | 💡 | 팁·노하우 |
  | report | 📋 | 보고서·리포트 |

- **캐시 시스템**: 동일 videoId 재분석 시 저장된 결과 즉시 반환 (API 비용 절감)
- 분석 결과 Firestore `summaries` 컬렉션에 자동 저장

### 2-2. 음성 녹음 분석

- 홈 화면에서 **즉시 녹음** 또는 **오디오 파일 업로드** (무료 회원 최대 20MB / 어드민 무제한 / TODO: 유료Pro 100MB)
- 지원 형식: webm, mp3, m4a, ogg 등 audio/* MIME 타입 전체
- Gemini 멀티모달로 음성 전사(STT) + 제목 자동 생성
- 전사 텍스트를 Claude 파이프라인으로 넘겨 **10개 카테고리 자동 분류 + 요약**
- 분류된 카테고리 색상의 **SVG 썸네일 자동 생성** (🎙 아이콘 포함)
- 분석 완료 후 로그인 사용자의 마이페이지에 **비공개로 자동 저장**

### 2-3. PDF 문서 분석

- 홈 화면에서 PDF 파일 업로드 (무료 회원 최대 20MB / 어드민 무제한 / TODO: 유료Pro 100MB)
- Gemini로 PDF 전체 텍스트 추출 (텍스트 기반 + 이미지/스캔 PDF OCR 모두 처리)
- Claude 파이프라인으로 카테고리 분류 + 요약 + 리포트 요약 생성
- 분류된 카테고리 색상의 **SVG 썸네일 자동 생성** (📄 아이콘 포함)
- 분석 완료 후 로그인 사용자의 마이페이지에 **비공개로 자동 저장**

### 2-4. 카테고리 재분석 (Re-analyze)

- 결과 페이지에서 다른 카테고리로 즉시 전환 분석
- 유튜브 영상: `/api/summarize` 재호출
- 음성·PDF: `/api/reanalyze-transcript` — 원본 전사 텍스트 재활용 (재전사 없음)
- 새 sessionId 발급 + Firestore 저장

## 3. 결과 페이지 (Result)

### 3-1. 영상 플레이어

- YouTube IFrame Player 임베드
- 자동재생·일시정지·시간이동 지원
- 타임스탬프 클릭 시 해당 시점으로 점프

### 3-2. 카테고리별 AI 요약 레이아웃

각 카테고리마다 **특화된 구조**로 요약을 표시합니다:

| 카테고리 | 표시 항목 |
|---|---|
| **recipe** | 완성 이미지, 재료 목록, 단계별 타임스탬프 + 설명, 난이도, 조리 시간 |
| **english** | 핵심 표현, 어휘, 문법 패턴, 문화 맥락 |
| **learning** | 핵심 개념, 주요 포인트, 예시 + 타임스탬프 |
| **news** | 헤드라인, 3줄 요약, 5W1H 분석, 시사점 |
| **selfdev** | 핵심 메시지, 인사이트, 실천 체크리스트, 명언 |
| **travel** | 여행지, 장소 목록, 동선, 실용 정보, 주의사항 |
| **story** | 등장인물, 타임라인, 장르, 결말 분석 |
| **tips** | 번호 매긴 팁 목록, 도구·준비물, Top 3 하이라이트 |
| **report** | 목차, 섹션별 정리, 결론 |

### 3-3. 도슨트 채팅 (Docent Chat)

- 결과 페이지 우하단 플로팅 버튼으로 상시 접근
- 현재 요약 콘텐츠를 컨텍스트로 멀티턴 대화
- "이 영상에서 가장 중요한 부분이 뭐야?", "3단계를 더 쉽게 설명해줘" 등 자유 질문

### 3-4. 세그먼트 댓글 (Segment Comment)

- 요약 내 특정 단락·단계에 말풍선 형태로 댓글 등록
- 등록된 세그먼트 댓글은 **하단 전체 댓글 목록에도 자동 반영** (📌 배지 포함)
- 댓글 목록에서 단락명 클릭 시 해당 세그먼트로 스크롤

### 3-5. 전체 댓글 섹션

- 루트 댓글 + 대댓글 (threaded)
- 학생 댓글 제출 시 교사 대시보드에 **활동 로그 자동 기록**
- 댓글 수 표시

### 3-6. 저장 및 공유

- **라이브러리에 저장**: 폴더 선택 + 공개 여부(비공개/공개) 설정 후 저장
- **스퀘어에 공개 발행**: 커뮤니티 피드에 등록
- **링크 복사**: 결과 페이지 URL 클립보드 복사
- **SNS 공유**: OG 태그 자동 설정으로 카카오·트위터 등 미리보기 지원

### 3-7. 번역

- 요약 내 특정 섹션을 원하는 언어로 번역
- 번역 버튼 클릭 → 인라인 번역 표시

### 3-8. 음성 컨트롤 (요리 영상 전용)

요리 영상 결과 페이지 하단 좌측 고정 마이크 버튼:

- **마이크 클릭 한 번** → 페이지 나갈 때까지 계속 켜짐 (자동 연속 인식)
- **명령 실행 후 자동 재시작** — 매번 탭할 필요 없음
- 에코 캔슬링(`echoCancellation`, `noiseSuppression`) 명시 적용
- 영상이 스피커로 재생 중이어도 오인식 최소화

**지원 음성 명령:**

| 명령 예시 | 동작 |
|---|---|
| "3단계", "세 단계로 이동" | 해당 단계 타임스탬프로 이동 |
| "10초 뒤로", "30초 앞으로" | 현재 위치에서 N초 이동 |
| "처음부터", "처음으로" | 영상 처음으로 이동 후 재생 |
| "재생", "틀어", "계속" | 영상 재생 |
| "멈춰", "일시정지", "스톱" | 영상 일시정지 |
| "빠르게", "빨리" | 배속 +0.25 |
| "느리게", "천천히" | 배속 -0.25 |
| "1.5배속", "2배속" | 해당 배속으로 직접 지정 |
| "보통", "정상 속도" | 1.0배속으로 복원 |

## 4. 마이페이지 / 개인 라이브러리

### 4-1. 프로필 카드

- 아바타 (이모지 선택 또는 사진 업로드)
- 이름, 이메일, 역할 배지 (일반·교사·학생)
- 토큰 잔액 표시
- 프로필 완성 유도 인센티브
- 교사로 전환 / 회원 탈퇴

### 4-2. 폴더 관리

- **생성·이름 변경·삭제**
- **3단계 공개 설정**: 비공개 / 친구 공개 / 전체 공개
- **공유 링크 생성**: 폴더별 고유 shareId → URL 복사
- 폴더 내 요약 **드래그 앤 드롭 재정렬**

### 4-3. 저장된 요약 관리

- 마순리 그리드 레이아웃
- **카테고리 필터** (전체·recipe·english·... 10개)
- **NLP 전문 검색**: 키워드 + 벡터 유사도 하이브리드
- 요약 이동 (다른 폴더로)
- 요약 삭제
- 태그·폴더 정보 표시

### 4-4. 서브 탭

| 탭 | 내용 |
|---|---|
| 라이브러리 | 폴더별 저장 요약 전체 |
| 친구 | 친구 요청·수락·거절·목록 |
| 여행 | 저장된 여행지(위시리스트) + AI 생성 일정 |
| 블로그 초안 | SEO 블로그 아티클 목록 |
| 숏츠 스크립트 | 추출된 숏폼 클립 목록 |
| 북마크 | 타임스탬프 메모 목록 |

## 5. 스퀘어 (Square) — 커뮤니티 피드

### 5-1. 피드 표시

- 마순리 그리드 (데스크톱 4열 / 태블릿 3열 / 모바일 2열)
- 카드마다: 썸네일, 카테고리 배지, 제목, 채널명, 좋아요·조회수
- 카드 13개마다 Google AdSense 광고 슬롯 삽입
- AI 기반 추천 카드 (사용자 좋아요 카테고리 기반, 없으면 인기 콘텐츠)

### 5-2. 필터 및 정렬

- **카테고리 필터**: 전체·recipe·english·learning·news·selfdev·travel·story·tips
- **정렬**: 최신순 / 인기순(좋아요) / 조회수순
- **자연어 검색** (NLP)

### 5-3. 카드 액션

- 좋아요 / 좋아요 취소 (카운트 실시간 반영)
- 조회수 자동 집계
- 작성자에게 메시지 보내기 (본인 제외)
- 본인 발행 요약 삭제

## 6. 클래스룸 — 교사·학생 학습 관리

### 6-1. 클래스 생성 (교사)

- 학교명·학년·반 입력 → **6자리 고유 클래스 코드** 자동 발급
- 마스터 폴더 자동 생성 (교사 라이브러리와 연동)
- 코드 공유로 학생 초대

### 6-2. 학생 등록

- 클래스 코드 + 이름으로 가입
- 전용 이메일 자동 발급: `{classCode}.{nameBase64}@cls.ssoktube.com`
- 가입 시 교사 마스터 폴더 자동 상속

### 6-3. 교사 대시보드

- **전체 통계**: 총 학생 수, 활동량, 퀴즈 성취도
- **학생별 상세 로그**:
  - 로그인 / 로그아웃 이력
  - 영상 시청 (시작·완료·시청 시간·진행률)
  - 퀴즈 시도 횟수·정답률
  - 이해도 마커: 완료(✅) / 헷갈림(🤔) / 모름(❓) 횟수
  - 댓글 제출 이력
- **영상별 분석**: 어떤 영상을 누가 얼마나 봤는지 집계
- **비디오 배포**: 교사 폴더의 영상을 학생 라이브러리로 푸시

### 6-4. 학생 경험

- 교사가 배정한 영상 열람
- 이해도 마커 클릭 (완료·헷갈림·모름)
- 퀴즈 응시
- 세그먼트·전체 댓글 작성 (교사 대시보드에 자동 기록)
- 개인 라이브러리 별도 유지

### 6-5. 활동 자동 로깅

`/api/classroom/activity` 엔드포인트로 다음 이벤트 자동 기록:

| 이벤트 타입 | 기록 내용 |
|---|---|
| `login` | 접속 시각 |
| `logout` | 로그아웃 시각 |
| `watch_start` | 영상 시청 시작 |
| `watch_complete` | 시청 완료 |
| `watch_progress` | 시청 진행률(%) |
| `quiz` | 정답 여부, 점수 |
| `meta` | 이해도 마커 레벨 |
| `comment` | 댓글 텍스트 |

## 7. 콘텐츠 생성 도구

### 7-1. 세그먼트 퀴즈

- english·learning 카테고리 결과 페이지에서 활성화
- 플래시카드 + 객관식 문제 자동 생성
- 학생 응시 시 교사 대시보드에 결과 자동 기록

### 7-2. 영어 워크시트

- 영어 영상 결과 페이지에서 생성
- **3가지 수준**: 초급(CEFR A1-A2) / 중급(B1) / 고급(B2+)
- 항목: 어휘 목록, 매칭 문제, 빈칸 채우기, 번역 문제
- PDF 다운로드 가능

### 7-3. 블로그 초안 생성

- 요약을 SEO 최적화 블로그 아티클로 자동 변환
- 항목: 제목, 도입부, 본문 섹션, 결론, SEO 키워드
- Markdown 형식 저장 → 마이페이지 "블로그 초안" 탭에서 관리

### 7-4. 숏츠 스크립트 추출

- 자막 전문 분석으로 **3~5개 숏폼 클립 구간 추천**
- 각 클립: 시작/종료 시각, 스크립트, 제목, 훅 이유, 유형(hook/tip/highlight/emotion), 초 단위 길이
- 전체 영상 편집 팁 포함
- 마이페이지 "숏츠 스크립트" 탭에서 관리

### 7-5. 여행 일정 생성 (AI 플래너)

- 마이페이지 여행 위시리스트의 저장 장소를 기반으로 AI 일정 생성
- **모드 선택**: 저장 장소만 / AI 추천 장소 포함
- 입력: 지역, 날짜 범위, 숙소, 도착 시각
- 출력: 시간대별 세부 일정 + 이동 팁
- 마이페이지 "여행" 탭에서 관리

### 7-6. 멀티 리포트 (다중 영상 분석)

- 1~5개 저장 요약 선택 + 리포트 제목·목적 입력
- AI가 영상 간 교차 분석 생성:
  - 종합 요약, 핵심 테마, 인사이트(출처 영상 인용), 공통점·차이점, 우선순위별 액션 아이템, 결론
- PDF 다운로드
